# Employee-Management


## Overview
A Java console-based application to manage employee records using OOP, Collections Framework, Custom Exceptions, Java 8 Optional, and Stream API.

## Features
- Add Employee
- View Employees
- Search Employee by ID
- Delete Employee
- Find Highest Salary Employee
- Filter Employees by Department

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Custom Exceptions
- Java 8 Stream API
- Optional

## Project Structure

```text
Employee.java
EmployeeService.java
EmployeeNotFoundException.java
Main.java
```

## How to Run

Compile:

```bash
javac *.java
```

Run:

```bash
java Main
```

## Concepts Covered
- Encapsulation
- Collections Framework
- Exception Handling
- Custom Exceptions
- Stream API
- Optional
- Method References
- Switch Case Menu-Driven Program
